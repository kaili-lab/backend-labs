# 补充资料

这里用于存放某个知识点的深入理解、参数级说明、踩坑记录和外部资料整理。

## 这一类文档的职责
- 补充 `docs/topics/` 中不适合展开过细的内容
- 记录具体实现细节、参数解释、经验总结和常见误区
- 为后续实验和主线设计提供更细的背景材料

## 适合放在这里的内容
- JDK 线程池各参数详解
- 线程池常见误用与踩坑
- 数据库连接池参数说明
- Redis 使用注意事项
- 消息队列重试、幂等相关经验总结

## 不适合放在这里的内容
- 完整的系统演化主线
- 只讲“什么时候用”的简要知识点说明
- 需要 runnable code 才能观察的实验内容

## 和其他目录的关系
- `docs/topics/` 负责说明“是什么、为什么、什么时候用”
- `docs/resources/` 负责说明“细节怎么理解、参数怎么配、常见坑是什么”
- 如果需要观察行为，应继续链接到 `labs/`
